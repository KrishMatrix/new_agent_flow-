# Agnes — Pitch Script

## Opening (30 seconds)
"Every e-commerce manager drowns in spreadsheets trying to figure out when to reorder, what's selling, and what's going wrong. Meet Agnes — your AI supply chain manager."

## Demo Flow (3 minutes)

### 1. Introduction
- Show the landing screen with three agent characters
- "Agnes coordinates three specialized AI agents, each with their own personality and expertise."

### 2. Forecaster Demo
- Type: "What's the demand forecast for blue sneakers next week?"
- Watch the Forecaster character walk in
- Show the inline chart with predictions
- "The Forecaster uses exponential smoothing on 90 days of sales data to predict demand, accounting for trends and seasonality."

### 3. Scout Demo
- Type: "Are there any anomalies in our data?"
- Watch the Scout character take over
- Show the anomaly alerts table
- "The Scout detected that red t-shirt returns spiked 3x and navy polo shirt is critically low on stock. Z-score analysis catches what humans miss."

### 4. Optimizer Demo
- Type: "When should I reorder red t-shirts?"
- Watch the Optimizer calculate
- Show the metrics table with reorder recommendations
- "The Optimizer calculated safety stock, reorder points using EOQ, and tells you exactly how many units to order and by when."

### 5. Dashboard
- Point to the side panel
- "Real-time KPIs, sales trends, top products, and low stock alerts — always visible."

## Closing (30 seconds)
"Agnes isn't just a chatbot — it's three specialized AI agents working together, each with real analytical tools behind them. Built in 24 hours with FastAPI, React, and Claude."

## Key Differentiators
- Multi-agent architecture with distinct personalities
- Real analytical tools (not just LLM responses)
- Mario-style character animation for engagement
- Inline charts and data tables in chat
- Works with or without an LLM API key (fallback mode)
